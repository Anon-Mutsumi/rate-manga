# Rate Manga
Give a rating to a specific manga only once, rather than doing it over and over on different websites

# Description
Tired of having to go though multiple sites to rate the same manga over and over? Rate Manga was created to save you from the monotonous task by only having you rate your desired manga only once.

# How to run Rate Manga
1. Configure the credentials.py folder by providing your credentials
2. Navigate to the folder of your local copy of this repository
3. Type "python ./rate_manga.py" into the command line, and hit Enter
